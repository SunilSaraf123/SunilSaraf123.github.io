Project Description:

 

UUP essentially is a data broker that aggregates and presents unified views of user data as a single XML document to client applications by brokering data in real-time from data sources that contain user data. The user data in the underlying data sources could be accessed via different protocols such as SOAP/XML, HTTP, LDAP, Relational and others. All of this are normalized into XML and presented via a single XML virtual view with a generic data agnostic API that allows performing C.R.U.D (Create, Read, Update, and Delete) operations on the virtual view.
